## üì¶ Installation & Setup

### Clone the repo:
```
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

### Install dependencies:
```
npm install
# or
yarn install
```

### Run the development server:
```
npm run dev
# or
yarn dev
```

### Build for production:
```
npm run build
```

### Preview production build:
```
npm run preview
```
